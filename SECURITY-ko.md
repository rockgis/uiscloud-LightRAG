# 보안 문제 신고

LightRAG 팀과 커뮤니티는 보안 버그를 진지하게 받아들입니다. 책임감 있는 취약점 공개 노력에 감사드리며, 기여에 최대한 감사를 표하겠습니다.

보안 문제를 신고하려면 GitHub 보안 권고를 이용하세요: [취약점 신고하기](https://github.com/HKUDS/LightRAG/security/advisories/new)

LightRAG 팀은 보고서 처리의 다음 단계를 안내하는 답변을 보낼 것입니다. 초기 답변 이후, 보안팀은 수정 및 공식 발표 진행 상황을 지속적으로 알려드리며, 추가 정보나 안내를 요청할 수 있습니다.

제3자 모듈의 보안 버그는 해당 모듈을 관리하는 담당자 또는 팀에 신고하세요.
